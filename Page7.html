Синтаксис оператора else
Оператор if может использоваться вместе с оператором  else , который будет выполняться, если условие, сформулированное в заголовке оператора  if, ложно.
Синтаксис оператора  else:
При выполнении программы прверяется условие, если оно истинно, выполняется код в теле оператора  if , если ложно - код в теле оператора  else.
Фигурные скобки могут быть опущены, если тело операторов if/else состоит из одного оператора.
